## 1. 핵심 컨셉 — AI 비서 마스코트 "캔들이"(가칭)

게임 전반에서 **내 옆의 AI 비서**가 코칭해주는 느낌을 준다. 촛불 캐릭터가 상황에 맞는
**표정 이모지 + 짧은 한마디(말풍선)**로 등장.

- **아바타**: 이모지 조합 기반. 기본 🕯️ + 상태별 표정(예: 😊 응원 / 🤔 신중 / 🎉 축하 / 😮 주의 / 🔥 열정).
  → 실제로는 `Mascot` 컴포넌트가 상태값(`cheer|think|warn|celebrate|idle`)을 받아 이모지·색을 스왑.
- **말풍선(`SpeechBubble`)**: 아바타 + **크고 짧은 한 줄**(핵심 키워드 위주, 12~20자 목표) + 꼬리.
- **톤**: 다정하고 짧게. 예) "지금은 관망이 편해요 🤔", "추세 살아있어요, 조금 더 볼까요?"
- **페르소나 규칙**: 훈수/명령 X → 관찰·제안형. 투자권유 표현 배제(학습 톤 유지).

> **오픈 이슈 A**: 캐릭터 이름(캔들이/초비/양초비서 등)과 말투(존댓말 통일 vs 반말 섞기) — 확정 필요.

## 2. 화면별 변경 (요청 #1~#5 매핑)

### 2.1 홈 (HomeScreen) — #1, #5
- **하단 "🏆 오늘의 랭킹" 버튼 제거.** 대신:
  - 상단 헤더의 **페이북겜머니 pill을 탭 가능**하게 → 랭킹 이동.
  - 데일리 카드의 **"오늘 N위" 칩을 탭** → 랭킹 이동.
- 마스코트가 인사/오늘의 한마디로 등장(예: "오늘도 100만원으로 시작! 🕯️").
- 톤: 라운드·파스텔 포인트, 아기자기한 마이크로카피.

### 2.2 플레이(턴 중) (PlayScreen) — #3, #5
- 기존 **매턴 팁(2초 배너)** → **마스코트 말풍선**으로 교체. 크고 짧게, 표정 포함.
- 초보자 **튜토리얼 코치마크**에도 마스코트 아바타를 붙여 "AI 비서가 알려주는" 톤 강화.
- 매수/공매도 `?` 미니설명도 말풍선 톤으로 통일.

### 2.3 결과/복기 (ResultScreen) — #2, #3, #4
- **차트 확대**: 리빌 차트 높이 200 → **크게(예 280~320)**, 화면 비중 상향.
- **마커 명확화**(`chartDraw.ts` reveal 모드 강화):
  - **내 진입 시점**(▲ 매수 / ▼ 공매도)·**청산 시점** 라벨/색으로 표시.
  - **최선의 한 수(best)** 구간을 별도 색/점선 + "여기서 샀다면" 라벨로 강조.
- **코칭 카드(좋았던 점 / 주의할 점 / 최선의 한 수)**: **마스코트가 말하는 형태**로,
  **큰 폰트 + 짧은 문장**(키워드 중심)으로 축약.
- **겜머니·"오늘 N위" 리워드 pill을 탭** → 랭킹 이동(#1과 동일 규칙).
- **'한 판 더' 자동 팝업(#4)** — 결과 렌더 직후(폭죽 등 연출 후) 자동 노출:
  - **무료판 남음** → "한 판 더 할까요?" · **예** = 바로 다음 무료판 / **아니오** = 홈 / **닫기(X·바깥탭)** = 결과 유지.
  - **무료 소진** → "광고 보고 한 판 더?" · **예** = 광고 → 게임 / **아니오** = 홈 / **닫기** = 결과 유지.
  - 마스코트가 물어보는 말풍선형 팝업.
- **복귀 경로 상시 노출**: 팝업을 닫아도 길을 잃지 않도록 결과 화면에 **[홈] · [랭킹]** 버튼 상시 유지.

### 2.4 랭킹 (LeaderboardScreen) — #1
- 진입 경로를 홈·결과의 **탭 연결로 일원화**(별도 버튼 제거). 화면 자체 구성은 유지(상금 스트립/뱃지).
- "처음으로 / 한 판 더" 하단 액션은 유지(랭킹에서의 이탈 경로).

### 2.5 전체 톤 — #5
- 초보자 친화: 둥근 모서리·부드러운 색·큰 글씨·짧은 문구·마스코트 상시.
- **교육 요소는 유지**: SIGNAL 점수, 지표 칩, 리빌(정답 공개), 최선의 한 수, 복기 카드.

## 5. 단계별 실행안 (승인 후)

1. **Phase 1 — 마스코트 시스템**: `Mascot`/`SpeechBubble` + 톤 스타일 토대.
2. **Phase 2 — 플레이 화면**: 매턴 팁·튜토리얼 마스코트화.
3. **Phase 3 — 결과 화면**: 차트 확대·마커, 코칭 카드, 리워드 탭 연결.
4. **Phase 4 — '한 판 더' 팝업(#4)** + 복귀 버튼 상시화.
5. **Phase 5 — 홈/랭킹 정리(#1)**: 버튼 제거·탭 연결.
6. **Phase 6 — 검증·문서·백업·배포**: 타입체크/빌드/헤드리스 스모크, README·본 문서 갱신, v0.13 백업.

각 Phase는 독립 배포 가능한 단위로 쪼개 리스크를 낮춘다.

## 6. 남은 확인 필요(오픈 이슈)

- **A. 캐릭터 이름·말투**: 이름(예: 캔들이) / 존댓말 통일 vs 반말 혼용.
- **B. '한 판 더' 팝업 노출 빈도**: 매 결과마다 vs 첫 1회만(피로도 고려).
- **C. 무료판 '아니오' 목적지**: 홈 고정 vs "게임 진입했던 화면"(홈/랭킹 등 진입점 기억).
- **D. 결산 시각(현재 새벽 6시)**: 프로토는 브라우저 로컬시간 → 실서비스는 **서버 KST** 기준 필요.
- **E. 버전 표기**: 본 리뉴얼을 **v0.14**로 진행할지.
